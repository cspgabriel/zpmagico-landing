Este NÃO é a API Oficial do WhatsApp

O Zap Mágico é uma ferramenta independente que automatiza o WhatsApp Web para pequenas empresas, autônomos e empreendedores. Não violamos termos — operamos como extensão do WhatsApp Web, igual outras ferramentas do tipo. Recomendamos a API oficial do WhatsApp para empresas de médio/grande porte.

Perguntas Frequentes

O Zap Mágico é gratuito mesmo?+
Sim. O download e o uso do app são gratuitos. Teste por 1 semana sem compromisso.
Preciso instalar algo no meu computador?+
Sim, você baixa o instalador .exe e instala no Windows. Leva menos de 2 minutos.
Como funciona a proteção contra bloqueio?+
No Disparo em Massa você configura delay aleatório entre mensagens, pausas a cada X envios e um limite diário por campanha — reduz o risco de bloqueio, mas não existe garantia de 0% de risco em nenhuma ferramenta desse tipo.
Quantas contas posso conectar?+
Você pode cadastrar mais de uma conta Zappy no app e escolher qual delas dispara cada campanha, além da conexão por QR Code nativo.
O app tem IA de atendimento automático?+
Não. O que existe é a integração com agentes de IA (Claude, ChatGPT etc.) via endpoint local — você usa um assistente de IA externo para disparar mensagens pelo app, mas não há chatbot de atendimento embutido.
Funciona em Mac ou Linux?+
Não. O instalador atual é apenas para Windows 10/11 (64-bit).
💻 Requisitos do Sistema

💻 Computador Windows

  • Sistema Operacional: Windows 10 ou Windows 11 (64-bit)
  • Hardware: Mínimo de 4GB de memória RAM e 500MB de espaço livre
  • Executável leve (.exe) otimizado que roda em segundo plano
  • Compatibilidade: Não funciona nativamente em macOS, Linux ou celulares

🌐 Conexão e WhatsApp

  • Conexão de Internet ativa e estável durante os disparos
  • Aparelho celular com WhatsApp ativo para leitura inicial do QR Code
  • O computador precisa estar ligado para que os envios sejam processados
  • Funciona com números novos ou já existentes no WhatsApp
